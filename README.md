# Лабораторные работы PHP

## Задание 1: Базовые задачи PHP

### Структура файлов
```
/php_basics/
├── calculator.php     - калькулятор с базовыми операциями
├── file_upload.php    - загрузка файлов на сервер
└── php_tasks.php      - решение алгоритмических задач
```

### Особенности

**calculator.php:**
- Калькулятор с четырьмя операциями (+, -, ×, ÷)
- Проверка деления на ноль
- Сохранение введенных чисел после операции

**file_upload.php:**
- Загрузка файлов через форму
- Создание папки uploads автоматически
- Отображение информации о загруженном файле (имя, размер, тип)

**php_tasks.php:**
- Рекурсивное возведение в степень
- Склонение времени (час/часа/часов, минута/минуты/минут)
- Вывод чисел делящихся на 3 (while)
- Четные и нечетные числа (do while)
- Вывод чисел без тела цикла (for)
- Многомерный массив областей и городов
- Фильтрация городов на букву "К"
- Транслитерация кириллицы в латиницу
- Транслитерация для URL с заменой пробелов

## Задание 2: Система авторизации с отслеживанием последних посещений

### Структура файлов
```
Lab7_PHP_2/
├── index.php          - перенаправление пользователей
├── login.php          - вход в систему
├── reg.php            - регистрация
├── logout.php         - выход
├── a.php              - страница А (Краснодар)
├── b.php              - страница Б (Авиабилеты)
├── style1.css         - стили для страницы А
├── style2.css         - стили для страницы Б
└── users.json         - данные пользователей (создается автоматически)
```

### Особенности
- Неавторизованные пользователи перенаправляются на страницу входа
- Авторизованные попадают на последнюю посещенную страницу (А или Б)
- Пароли хранятся в зашифрованном виде
- Отслеживание через PHP сессии

## Задание 3: Выбор стилей через Cookies

### Структура файлов
```
Lab7_PHP_3/
├── index.php          - главная страница
├── setting.php        - выбор стиля
├── style1.css         - синий градиент
├── style2.css         - оранжевый градиент
└── style3.css         - темный градиент
```

### Особенности
- Три разных стиля оформления на выбор
- Выбор сохраняется в cookies на 30 дней
- Применяется ко всем страницам автоматически
- Минималистичный дизайн для демонстрации

## Задание 4: Список группы из Excel

### Структура файлов
```
/Lab7_PHP_4/
├── simplexlsx.php     - библиотека для работы с xlsx
├── groups.php         - главная страница с таблицей
└── group_list.xlsx     - список группы (CSV формат)
```

### Особенности
- Чтение данных из Excel (xlsx) или CSV файлов
- Кнопка "Показать список" для загрузки данных
- Фильтрация по ФИО (регистронезависимая)
- Подсчет количества студентов
- Поддержка обоих форматов: xlsx и csv

---

## Общие требования

- PHP 7.0+
- Браузер с поддержкой cookies
- Для xlsx: расширение ZipArchive
